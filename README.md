# VK to Yandex Disk Uploader

Учебный Python-скрипт для загрузки фотографий пользователя из VK на Яндекс.Диск.

## Что делает проект
- получает фотографии из профиля VK через VK API
- выбирает фотографии максимального размера
- загружает 5 фотографий на Яндекс.Диск
- сохраняет информацию о загруженных файлах в JSON

## Используемые технологии
- Python
- requests
- работа с REST API (VK API, Yandex Disk API)

Проект выполнен в учебных целях.
